# Gerador-de-senha-aleatoria-mobile

Aplicativo mobile desenvolvido para gerar senhas seguras de forma rápida e simples.  
O usuário pode personalizar o tamanho da senha e copiar facilmente para uso em outros aplicativos.
O usuário tambem tem acesso a uma tela com todas as senhas no qual ele pode apagar as que desejar.

---

## Tecnologias utilizadas
- React Native
- Expo
- JavaScript

---

## Como funciona

O aplicativo utiliza uma string com letras maiúsculas, minúsculas e números.
selecionando caracteres aleatórios de acordo com o tamanho definido pelo usuário.

---

 ## Como executar o projeto

 ### Pré-requisitos
- Node.js instalado
- Expo CLI instalado
- Git

### Passo a passo

```bash
# Clone o repositório
git clone https://github.com/Pedrofcaldas07/Gerador-de-senha-aleatoria-mobile.git

# Entre na pasta
cd Gerador-de-senha-aleatoria-mobile

# Instale as dependências
npm install
# ou
yarn install

# Execute o projeto
npx expo start
